# Concepts

Use this area for notebook-first learning.

Suggested categories:

- `math/`: derivatives, integrals, linear algebra, probability
- `machine_learning/`: algorithms, losses, metrics, validation
- `deep_learning/`: backprop, activations, CNNs, transformers
- `llms/`: tokenization, attention, generation, RAG, agents

Keep each notebook small. One notebook should teach one concept.
